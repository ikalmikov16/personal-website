# IkaOS

A faithful Mac OS X Snow Leopard replica built as a personal portfolio. Explore projects, resume, and links through a fully interactive browser-based desktop experience — complete with draggable windows, a dock, menu bar, Spotlight search, Stickies, and more.

On mobile, visitors get an era-appropriate **iPhone OS 3** home screen with the same content.

## Features

- **Desktop OS** — Menu bar, dock with magnification, draggable/resizable windows, genie minimize animation
- **Apps** — Finder, Safari, Terminal, System Preferences, Stickies, Preview (resume PDF)
- **Spotlight Search** — Ctrl+Space to search apps, projects, and links
- **App Switcher** — Option+Tab to cycle between open windows
- **About This Mac** — Classic info dialog from the Apple menu
- **Context Menus** — Right-click on desktop, dock icons, and sticky notes
- **Boot Screen** — Apple logo + progress bar on first visit
- **Mobile** — iPhone OS 3 lock screen, home screen with glossy icons, and full app views
- **Wallpapers** — Change the desktop background from System Preferences
- **Easter Egg** — Try the Konami code

## Run locally

```bash
bun install
bun run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
bun run build
bun run preview
```

## Stack

- [React](https://react.dev) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev) + [Bun](https://bun.sh)
- CSS (no frameworks, no backend)
